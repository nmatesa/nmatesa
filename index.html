HTML Forms

Form Elements

Tag Name Description
<form> Form collect information in input elements and send via action and method attribute
<input> Input render a variety of inputs with type attributes like text, checkbox, button..
<select> Select create a dropdown list with only one option can be selected
<datalist> Datalist multiple option elements but also allow to write alternative inputs
<textarea> Textarea text-box for multi-line input with rows and cols attributes
<label> Label identification with matching values of input id and label for attribute

Form Attributes

Attribute Description
name for input to send data it needs key-value pairs where only name need to be specified
required specifies that the field must include a value
max validator for inputs of type number specifies the maximum value
min validator for inputs of type number specifies the minimum value
maxlength validator for input of type text specifies the maximum number of characters
minlength validator for input of type text specifies the minimum number of characters
pattern regular expression to match against the value of the input, i.e. pattern="[A-Za-z]{3}"

Input Types

Type Description
checkbox render a single checkbox; checkboxes with same topic should use the same name
number allow to enter only numbers
range range slider as a selector between minimum and maximum and can use step attribute
text single row to enter only text
radio only a single option can be chosen from a group of radio buttons with the same name
password single row to enter censored text
